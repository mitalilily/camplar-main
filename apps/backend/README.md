# Camplar Backend

This app contains the Camplar API server, scheduled jobs, integrations, and supporting scripts.

## Run Locally

```bash
npm install
npm run dev
```

## Notes

- Keep the current deployed domains and API endpoints unchanged unless you are doing a deliberate migration.
- Existing deployment files and service names are preserved so the current hosted setup is not disturbed.
- Use this app together with `apps/client` and `apps/admin` inside the monorepo.
